TEAM-PROJECT-SETS
------------------

The provided team projects sets are a simple way to check out the whole tacos platform with eclipse.
Just import the files using the import manager.

Description:
------------

Tacos_v2.psf -> current developing path with eclipse 3.4
Tacos_v1.psf -> the old version, running with eclipse 3.3
